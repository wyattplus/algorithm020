# 总结
关于10周的课程很快就结束了，但是其实对我来说，这才是这方面的开始。这70天是一个知识输入阶段，后续还需要更多的精力投入去完成自己的成长。
不可能70天就能完成一项技能的完美成长，这都需要时间积累的。
自己所恐惧的，需要自己去面对，这就是成长。加油！
